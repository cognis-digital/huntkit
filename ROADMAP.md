# SENTRYLOG — Roadmap

> Single-file SIEM for small teams — Sigma rules + multi-source ingest

## Now (v0.1.x)
- Stable CLI: `scan` with table / JSON / SARIF / HTML output.
- Bundled demo scenarios + smoke tests.
- MCP server for Cognis.Studio integration.

## Next (v0.2)
- Ship more upstream rule packs and ingest adapters.
- Streaming / near-real-time mode.
- Hardening, docs, and more demo scenarios.

## Later (v1.0)
- Performance pass, packaging to PyPI, and a stable rule/plugin API.
- Pro tier + commercial support (see `licensing@cognis.digital`).
